## Design / Layout Agent Instructions

iPadの縦横比は必ず元のまま保ち、拡大・縮小しても縦横比が変わったり歪んだりしないようにしてください。

画面上のすべての文字要素（タイトル、見出し、本文など）およびバッジは、iPad本体・スクリーンショット画像と一切重ならないようにレイアウトしてください。スクリーンショットがバッジにはみ出したりかぶったりしないこと。十分な余白を取り、端末・画像とテキスト・バッジが視覚的に分離されて見えるように配置してください。
